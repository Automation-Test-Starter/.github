# 🙋‍♀️ 简短介绍

组织因 QA 而生，也为 QA 服务

一个为 QA 提供自动化测试的起步项目，包含了常用的测试框架和工具，以及一些常用的测试用例。

目前包含了 API 自动化测试和性能测试的起步项目，后面也会陆续添加其他类型的自动化测试。
